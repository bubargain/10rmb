MODE 接口文档

get_wishlists

获取用户的所有心愿单

URL

服务器URL get_wishlists

支持格式

JSON

HTTP请求方式

GET / POST

是否需要登录

YES

访问授权限制

访问级别：普通接口
频次限制：是

请求参数

  必选 类型及范围 说明
amount true int 用户的wishlist总数
utime true string 用户的wishlist最后一次更新时间
lists
lists/wishlist_id true int wishlist id
comments true string 用户针对某一wishlist的评价
img_link true string wishlist九宫格图片
ctime true string wishlist创建时间

注意事项

无

返回字段说明

返回值字段 字段类型 字段说明
goods_id int 返回查询的商品ID
goods_title string 商品的标题描述
img_link string 商品图片链接
img_detail_link string 商品详情图片链接
price float 商品价格
size string 商品的size属性
color string 商品有几种颜色，以，号区隔，如#000，#3f4e5f,#xxxxx
has_coupon char 是否含有试用劵，Y or N
Coupon/coupon float 试用劵面额，单位美金
Coupon/amount int 试用劵总数量
Coupon/left_amount int 试用劵剩余数量
Coupon/coupon_id int 试用劵ID
Coupon/expired_time int 试用劵过期时间
ctime int 商品创建时间，时间戳类型
last_time int 商品有效时间，自创建时间开始计算，单位为秒
brand_id int 商品所属品牌ID
brand_id int 商品所属品牌名称

错误提示

返回值字段 字段类型 字段说明
error string 错误说明
