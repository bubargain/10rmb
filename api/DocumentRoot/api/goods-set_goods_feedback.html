MODE 接口文档

goods/set_goods_feedback

设置商品测试反馈

URL

服务器URL /goods/set_goods_feedback

支持格式

JSON

HTTP请求方式

GET / POST

是否需要登录

YES

访问授权限制

访问级别：普通接口
频次限制：是

请求参数

  必选 类型及范围 说明
goods_id true int 商品ID
user_id true int user id
fd true int 设置用户喜欢还是不喜欢该商品，值为”like“或”nope“

注意事项

无

返回字段说明

返回值字段 字段类型 字段说明
status string success 或 false
fd string 当前喜欢状态（确认服务器已经修改成功）

错误提示

返回值字段 字段类型 字段说明
error string 错误说明
