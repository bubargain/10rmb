MODE API 说明文档

用户接口
user/show_brands 获取用户关注的品牌列表
user/show_dialog 获取用户的心愿记录
user/show_profile 获取用户基本信息 （头像，昵称）
账号接口
account/verify_email 验证邮箱状态
account/login 登录验证
account/facebook_login Facebook登录
account/get_uid OAuth授权之后获取用户UID
account/reset_token 重置token过期时间
account/get_uinfo 获取用户基本信息
account/set_uinfo 设置用户基本信息 （包含密码）
account/get_privacy 获取隐私设置信息
商品接口
goods/get_goods 获取商品的基本信息
goods/set_goods_feedback 设置商品测试反馈
秀场接口
runway/get_new 新建一组秀场
心愿单接口
wishlist/get_wishlists 获取用户的wishlist
wishlist/get_wishlist_by_id 按ID获取用户wishlist
wishlist/get_newest_wishlist 获取用户最新的wishlist
wishlist/add_goods 向特定wishlist中加入商品
wishlist/del_goods 向特定wishlist中删除商品
订单接口
order/get_coupon 用户获取试用劵
order/set_coupon_consignee 用户填写试用劵消费人信息
order/get_orders 获取用户订单信息列表
提现接口
cash/apply_cash 用户发出提现申请
cash/set_credit 用户获得信用值
品牌接口
brand/get_fans_count 获取品牌粉丝数据
brand/set_like 用户关注/取消品牌
系统接口
sys/brand_list 获取活跃中的品牌名称列表
sys/occasion_list 获取场合列表
sys/style_list 获取商品风格列表
